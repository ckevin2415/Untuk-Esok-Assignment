# Untuk Esok Web Project

## Project Overview

Untuk Esok web project is a platform for a non-governmental organization (NGO) managed by high school students that serves as a space for the younger generation to actively contribute to society and support the Sustainable Development Goals (SDGs). This project is created as part of the WEBPAGE PUBLISHING (CSW202) course at Help University and a collaboration of students from ITB STIKOM Bali International Dual Degree Programme with Untuk Esok.

This project is built using core web technologies: HTML for structure, CSS for styling, and JavaScript for interactivity.
